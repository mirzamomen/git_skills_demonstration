# git_skills_demonstration
This is a simple repo to be used in git skills demonstrations

# store-demo

A tiny, deliberately trivial "corner store" script used for an in-class
Git demonstration. There is nothing to learn about the code itself —
`store.py` just holds a few constants, a `MENU` dict, an `INVENTORY`
dict, and a couple of small functions.

Run it with:

```
python3 store.py
```

## Usage

To see the available options, run:

```
python3 store.py --help
```

(or `python3 store.py -h`)

## Before your demonstration

Read `SCENARIO.md`. It describes a realistic situation you'll need to
resolve using Git — no command list is provided. You're expected to
know which commands to use.

Useful commands to have fresh in your mind beforehand: branching,
switching branches, staging, committing, merging, viewing diffs and
history, and resolving a merge conflict. You will not be given a
step-by-step script during the demonstration.
